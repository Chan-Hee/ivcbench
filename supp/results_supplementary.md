# Supplementary Results: immune-aware perturbation benchmark

This file holds the secondary detail for the main Results ("An Immune-Aware Benchmark for Perturbation Prediction"), including the per-unit detail of the two added models, CellOT (neural optimal transport) and chemCPA (native chemistry-aware CPA). Each note is keyed to its main-text subsection. All values are generated under leak-audited splits and are summarized in the main Tables 3 to 6 and Figure legends 2 to 6. The CellOT donor-transfer result revises the donor-axis verdict (a model-family-dependent outcome, as stated in the main text); chemCPA and CellOT on Kang corroborate the existing verdicts.

## S2. Benchmark design: baselines, applicability gate, and method funnel

Empirical floors and the comparator hierarchy. The four simple baselines are control-as-prediction, the training-mean (cell-mean) shift, a donor shift, and a denoised linear (PCA) shift. The universal two-member floor is {cell-mean shift, linear-PCA shift}, and a conditioned model must beat BOTH members to count as having delivered; control-as-prediction, the donor shift, and the training-mean shift are descriptive context comparators, not universal-floor members. Where the main tables additionally report the per-cell best of all four baselines, that quantity is a descriptive upper envelope only and is not the comparator used for any verdict. We call the baselines empirical floors because a useful conditioned model should exceed them, but the term is descriptive and does not assert a mathematical lower bound.

Replication unit and seeds. Hypothesis tests resample the biological unit of replication named per axis (immune lineage, dataset, or donor). Neural training seeds are collapsed to technical repeats within a unit before testing and are never counted as independent observations; where a seed range is reported (the Kang CD14⁺ monocyte row in the statistical-decision summary, and the Frangieh modality rows) it is a seed range, not a cluster confidence interval. Where three-seed replication or partition-robustness checks were run (scGen cell-context and modality; the linear knockout-embedding partition check), the seed-resolved values are deposited as dedicated tables in the code release rather than expanded into the primary single-seed results table, and the partition-robustness seed index denotes resampled data partitions rather than training-seed replicates.

Seed governance and inference primacy (one place). The final replicates of the two added models are the 3-seed CellOT runs (across-seed SD ±0.004 on the Soskic donor axis) and, for the Kang cell-context latent-shift result, the 3-seed scGen runs; within each biological unit the training seeds are collapsed to technical repeats and never enter the bootstrap as independent observations. Every reported confidence interval is a biological-unit cluster bootstrap (over lineages, datasets, or donors), not a bootstrap over seeds. This statement supersedes any "1 seed" wording in the per-cluster working manifests, which referred to a single deposited draw rather than the seed-aggregated final replicate used in the submitted results. The primary inference on each axis is the cluster-bootstrap confidence interval together with the Wilcoxon signed-rank p where reported; the "every resample positive/negative", "100% positive", and per-unit percent-positive figures are sign-consistency descriptors across the resampled biological units, not p-values and not themselves inferential tests.

AUCell-Δ coverage by gene set (expanding Table 4). Immune-program fidelity is computed wherever a curated transcriptional immune gene set is defined: the type-I-interferon set for Kang; T-cell activation, IL2-STAT5, and type-I and type-II IFN modules for Soskic; ISG, NF-κB, and effector programs for OP3; and five T-cell programs for the primary-T CRISPR collection. It is not defined for Frangieh, whose held readout is the CITE-seq surface-protein panel rather than a transcriptional program.

Method funnel (expanding Table 5). Ten conditioned entrants were executed across the surveyed model classes (of the roughly thirty surveyed). Two were added to close the model-family coverage gaps the benchmark flagged: a neural optimal-transport predictor (CellOT) for stimulation and donor transfer, and a native chemistry-aware compositional model (chemCPA) for unseen-compound prediction; both ran faithfully and leak-safe. CINEMA-OT remains a perturbation-agnostic distributional comparator, and the optimal-transport family is now also represented by the conditioned predictor CellOT. A surveyed method was not run in three situations: no maintained implementation was available; a split's inputs were undefined for a dataset (for example, a chemical structure for an unseen compound, or a gene-perturbation graph for a graph model); or the method does not apply to single-cell perturbation transfer. A model is reported as native where its published task matches the split, adapted where it was re-fit to a split outside its original design, and as a floor where only its non-conditioned reduction applies.

Operational definition of "adapted". A model is labelled adapted when its published architecture is left unchanged but it is re-fit per evaluation fold from the held-out fold's training cells only (for example CellOT and scGen on the donor and CRISPR/OP3 splits: the model is retrained within each leave-one-out fold using only training-fold cells, training-fold response genes, and a training-fold PCA basis, so no held-out information enters fitting). "Adapted" therefore denotes a leak-free per-fold re-fit of the published model to a split outside its original design target, not a modification of the model itself; "native" denotes that the published task matches the split with no per-fold re-fit beyond the authors' own procedure, and "floor" denotes that only the model's non-conditioned reduction is applicable. Adapted and floor entries are never reported as native capability.

## S3. Cell-context transfer (expanding "Cell-context transfer and immune-program recovery")

OP3 against the primary comparator. The FP-ridge advantage of +0.119 over the best simple baseline corresponds to +0.123 against the pre-specified training-mean/donor comparator, with the same direction and significance.

Kang per-seed detail. The CD14⁺ monocyte gap is reported as the three-seed mean rather than the optimistic single draw: scGen averages 0.884 over seeds {0,1,2} versus the donor-shift baseline 0.819, a mean gap of +0.065 (three-seed range +0.016 to +0.092). The previously deposited +0.098 single-draw sits at or above the three-seed maximum and overstates the seed-aggregated evidence; the donor shift is the binding comparator here because the training-mean baseline for this lineage is only 0.456. Across three training seeds the CD14⁺ monocyte and CD8⁺ T advantages were the only sign-robust ones (CD14⁺ monocyte positive in 3/3 seeds), but the gap is not a stable +0.10 and the n = 3 t-confidence-interval on it crosses zero, so this local advantage is under-powered (n = 3 seeds). This is a seed range, not a cluster confidence interval.

Program-versus-magnitude dissociation on OP3. The four tested conditioned families ranked the ISG gene set concordantly with the observed compound response (AUCell-Δ: CPA 0.79, scGen 0.78, FP-ridge 0.77, STATE 0.75). On response magnitude (Pearson-Δ), only FP-ridge cleared the linear-PCA magnitude floor of 0.27, reaching 0.39, while the latent and hybrid families fell below it (scGen 0.18; CPA and STATE near 0.05). The compound-matching permutation null permutes the correspondence between each compound's predicted and observed program-Δ while holding the per-compound distributions fixed (2,000 permutations); it gives a per-lineage one-sided p < 5 × 10⁻⁴ and a Fisher-combined p = 3.2 × 10⁻¹⁰ across the four coarse lineages used for the program analysis, supporting the recovery as compound-resolved for FP-ridge specifically and not for the other three families.

CellOT on Kang stimulation transfer. CellOT (neural optimal transport with f/g input-convex potentials; official bunnech/cellot, dedicated python-3.9 environment) was run on the Kang leave-one-lineage-out split as a task-matched stimulation-transfer model (train source = control cells of the non-held lineages, train target = the matching IFN-β cells; predict the held lineage's IFN-β response from its own control cells; seeds {0,1,2} collapsed within lineage; every fold leak-safe). Across the eight lineages the conditioned-minus-baseline Pearson-Δ was +0.012 (cluster-bootstrap CI [−0.017, +0.045]; 4/8 lineages positive; Wilcoxon p = 0.64), competitive with scGen (−0.016) and not separating from the matched primary baseline. CellOT was markedly stronger on distributional fidelity (E-distance gap +9.6, 8/8 lineages, p = 0.008) and modestly better on type-I-IFN program recovery (AUCell-Δ +0.055, 7/8 lineages), consistent with optimal transport optimising the perturbed distribution rather than the per-gene response direction. CellOT does not change the Kang local/competitive interpretation.

## S4. Perturbation extrapolation (expanding "Perturbation extrapolation")

CRISPR collection provenance. The five primary-human-T CRISPR Perturb-seq datasets are Shifrut (KO), Schmidt (CRISPRa), McCutcheon (CRISPRi and CRISPRa), and Chen (KO). Provenance note: accession GSE255832 is Pretto 2025, a mouse in-vivo CD8 metabolic screen that belongs to the modality/complex-context (Frangieh/C4) line of work and is not Chen; the Chen 2025 human FOXP3 Perturb-icCITE-seq data used here are at DDBJ PRJDB16517 / GEA E-GEAD-648. This corrects any cross-reference that conflated the two accessions.

Estimand and robustness for the CRISPR result. The training-mean shift was the best simple baseline in all 15 dataset-by-holdout cells, so the comparator is the pre-specified primary baseline rather than a post-hoc oracle. Leave-one-dataset-out held the gap between −0.22 and −0.26 with every held-out dataset negative. A complementary best-deep minus best-linear margin (the gap between the best deep/conditioned model and the best linear baseline on a split), re-derived in-frame following Ahlmann-Eltze, C., Huber, W. and Anders, S. (2025), "Deep-learning-based gene perturbation effect prediction does not yet outperform simple linear baselines," Nature Methods, reached the same conclusion across the 10%, 25%, and 50% unseen-gene holdout fractions, with mean margins of −0.24, −0.23, and −0.25; the negative result is therefore not specific to a single holdout fraction. The in-frame best-deep minus best-linear margin is re-derived in the repository by scripts/reviewer_batch4_analysis.py (the REQ 3 block) from the deposited per-split leaderboards results/C3/results_raw.csv (CRISPR unseen-gene) and results/C5/results_raw.csv (OP3 unseen-compound), so the margin computation is reproducible from the deposited values without re-running any model. Among individual conditioned models the graph methods trailed by roughly 0.28 to 0.30, while the foundation model scGPT and the published hybrid STATE sat near the control/null level. CINEMA-OT reproduces the empirical control-to-treated shift rather than predicting from the held-out gene, and is never counted as a conditioned win. Pooled per-model dataset averages (a different weighting from the paired per-cell estimand, and not subtractable against it) are: best single deep model AttentionPert 0.178, CINEMA-OT 0.426, pooled training-mean 0.457.

The failure is representational, not architecture-specific: a trivial nearest-gene prior also fails to beat the floor. To test whether the unseen-gene gap reflects the difficulty of representing a held-out perturbation from gene-side similarity — rather than a shortcoming peculiar to deep graph or foundation models — we added a parameter-free perturbation-representation control. For each held-out target gene we predict its effect as the observed training effect of its single nearest training gene, with the neighbour chosen two ways: (i) by a co-expression graph built on control cells only (gene–gene Pearson correlation across control-cell expression; never touching any perturbation effect, so leak-safe), and (ii) by GO-term Jaccard similarity using the cached gene2go annotation (data/_assets/gears/gene2go_all.pkl). Both priors are scored on the identical downstream-only Pearson-Δ, the identical leave-one-gene-out splits (10/25/50% holdout), and the identical biological units as the deep models, against the same universal floor; the harness reproduces the deposited cell-mean Pearson-Δ to within 6×10⁻¹⁷ across all 15 dataset-by-holdout cells (an exact-scoring anchor). Averaged over the 15 cells the nearest-gene co-expression prior reaches 0.223 and the GO-Jaccard prior 0.285 Pearson-Δ, both below the cell-mean floor (0.464) in 0 of 15 cells — the same shut-out the deep graph models post (GEARS 0.161, AttentionPert 0.183; 0/15 each). The simple gene-similarity priors in fact sit at or slightly above the deep models, so the inability to beat the training-mean shift on an unseen gene is a property of gene-side perturbation representation on these immune CRISPR data, not of any one model class. ## S5. Modality stress test (expanding "Modality stress test")

scGen and the linear knockout-embedding shift fell below the training-mean baseline in both modalities, with an RNA gap of approximately −0.05 to −0.13 and a protein gap of approximately −0.59 (scGen protein near 0.09); the verdict held across seeds {0, 1, 2}. This is a readout-modality stress test, RNA-trained models predicting a CITE-seq surface-protein readout from static CRISPR knockouts in tumour cells (not a time-resolved Perturb-seq perturbation), so the protein collapse (≈ −0.59) is specific to RNA-to-protein transfer on this single held-out dataset, has no independent replication, and is not a general modality-robustness verdict. scGen was trained for 40 epochs on this axis versus 60 on the others and was not re-run at 60 epochs, so the epoch difference remains a potential confounder for the scGen number specifically; the epoch-free linear knockout-embedding shift reproduces the same protein-readout collapse (around 0.08 at the 25% holdout), so the collapse is not attributable to under-training alone.

Per-marker sign-match in the CITE-seq panel (Figure 5c). PD-1 (CD279) was the best-recovered marker (sign-match 0.98) and the single largest immune effect in the panel (obsΔ ≈ −0.40, effect-z 1.79/1.90 at the 25%/50% holdouts). PD-L1 (CD274), by contrast, was essentially unperturbed at the surface: its observed delta is near zero and sign-unstable across holdouts (+0.042 at 25% but −0.023 at 50%), with effect-z only 0.28/0.09 (the smallest non-trivial in the panel), so its direction is not reliably recoverable (sign-match 0.355/0.460 ≈ chance). This is not a wrong-sign reversal of a real effect: there is no stable surface effect to recover, because the checkpoint ligand PD-L1 is post-transcriptionally buffered at the cell surface (CMTM6/SPOP stabilization; PD-L1 surface decoupled from cis transcription in Perturb-CITE-seq, Frangieh et al. 2021), so its surface level need not follow the transcriptomic shift. HLA class I was marker-dependent: HLA-E was moderately recovered (0.74) while HLA-A was weak and near-chance (0.65), and the two markers disagreed in response sign, so class-I recovery should not be read as uniform. CD49f and CD59 are likewise near-zero / sign-unstable for the same reason, so PD-L1 is not uniquely pathological; the predictable checkpoint signal is the receptor (PD-1), not the surface ligand (PD-L1). Whether recovery concentrates on the most immunologically informative markers would require a formal enrichment analysis and is not assessed here.

## S6. Donor generalization (expanding "Donor generalization and random-split optimism")

CellOT on Soskic donor transfer (the donor-axis main result). CellOT was run on the full leak-proof 106-donor leave-one-donor-out split (train source = 0 h cells of the non-held donors, train target = their 16 h cells; predict the held donor's 16 h response from its own 0 h cells; cap 300 cells per donor; three training seeds {0,1,2} collapsed within donor; every fold leak-safe; the per-donor primary baseline matched the scGen run in 100% of donors). It averaged 0.369 against the per-donor primary baseline of 0.267, a donor-bootstrap gap of +0.102 [+0.083, +0.122] with 87.7% of donors positive (p < 0.001), and the advantage held on the other two immune-aware metrics (E-distance gap +4.94, 100% of donors positive; AUCell-Δ gap +0.012, 96.2% positive; both p < 0.001). The latent-shift scGen, by contrast, averaged 0.144 for a gap of −0.123 [−0.137, −0.110] (1.9% positive). The donor-axis verdict is therefore model-level, not a family-level "optimal-transport wins donor" claim: the win is specific to CellOT, and the only available second optimal-transport data point points the other way. A deposited paired comparison on the shared donors (results/_paper/scpram_vs_cellot_donor_paired.csv, partial n = 61 of 106 as the second optimal-transport model scPRAM finalizes) shows scPRAM beating CellOT in only 1 of 61 donors, a mean per-donor gap of −0.191 (scPRAM 0.152 vs CellOT 0.343) with a Wilcoxon signed-rank p = 1.3 × 10⁻¹¹ — scPRAM is consistently and overwhelmingly below CellOT donor-for-donor, so the second optimal-transport model does not reproduce the win (scPRAM is unscored in the primary tables and finalizing to n = 106; the deposited n = 61 paired figure is preliminary). We therefore state the donor result at the model level: a CellOT-specific win, with the scored latent comparator scGen far below the matched baseline (−0.123 [−0.137, −0.110], 1.9% positive) and the one available second optimal-transport model (scPRAM, preliminary) losing. This rests on representation rather than family membership (optimal-transport geometry versus latent shift, suggestive not causal); CellOT is additionally adapted (re-fit per fold) on Soskic, and full within-family replicability awaits the finalized scPRAM n = 106 run and a second latent-shift model on this axis. 

Diagnostics and sensitivity. The donor result was computed under a deterministic 300-cell-per-donor cap, with the per-donor primary baseline matched to the scGen run in 100% of donors. The reported p < 0.001 is a Wilcoxon signed-rank test over the 106 per-donor conditioned-minus-baseline differences; the donor-bootstrap interval [+0.083, +0.122] (the canonical CellOT donor-axis interval reported in the main text and the statistical-decision table) is a 10,000-resample cluster bootstrap over donors. The E-distance and AUCell-Δ gaps are sign-oriented so that a positive value favours conditioning (E-distance is lower-better and was negated; AUCell-Δ is higher-better). The full 106-donor sweep was run for three training seeds {0,1,2}; per-donor predictions were averaged within donor before the bootstrap (matching the Kang treatment), and the per-seed donor-aggregated gaps were +0.0996, +0.100, and +0.107 (mean ± SD +0.102 ± 0.004), so the donor advantage is seed-stable and seed 0 is not an outlier. Downsampling-cap sensitivity and the relation of per-donor cell count to the per-donor gap were not separately swept; the per-donor and raw three-seed outputs from which a per-donor-cell-count sensitivity can be derived (cellot_soskic_by_donor.csv and cellot_soskic_raw_3seed.csv), with the run script at scripts/cellot_soskic.py, are available in the full archive / on request rather than shipped in this lightweight release. AUCell-Δ here is a negated program-shift error (lower error favours conditioning), whereas on the cell-context and perturbation axes AUCell-Δ is a program-shift correlation; both are oriented so that a positive value favours conditioning, and the AUCell score is a rank-based approximation.

Multi-metric detail for scGen on Soskic. With each metric oriented so that a positive value favours conditioning, the scGen donor-bootstrap differences were −0.123 [−0.137, −0.110] for response direction (Pearson-Δ), −0.056 [−0.089, −0.027] for distributional fidelity (E-distance), and −0.012 [−0.017, −0.006] for immune-program recovery (AUCell-Δ mean absolute error). All three lie below zero, so the latent-shift model fails on every metric, whereas CellOT (above) exceeds the baseline on all three.

Kang random-split optimism control. Random cell splits inflated Pearson-Δ by +0.017 at the observation level (paired Wilcoxon p = 2.2 × 10⁻³) and by a donor-unit cluster-bootstrap interval of [−0.0003, +0.033] over 8 donors, so random splits remain a small but suggestive optimism control.
